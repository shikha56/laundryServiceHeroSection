# Laundry Wallah / Laundry Mart

A simple and responsive **HTML webpage** for **Laundry Wallah / Laundry Mart**.  
This project is designed to present laundry services in a clean and user-friendly way, making it easy for customers to explore services, pricing, and contact details online.

## Features

- Clean design
- Service section to showcase laundry offerings
- Pricing section for transparent rates
- About section for information about laundry wallah / laundry mart
- Contact section with name, phone, or email
- Easy to customize HTML

## Technologies Used

- **HTML5** for webpage structure
